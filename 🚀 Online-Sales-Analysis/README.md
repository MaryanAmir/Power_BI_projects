# Online Sales Analysis Dashboard

## 📊  Project Overview

This project analyzes an online sales dataset using Power BI to understand product performance, regional sales distribution, and revenue trends.

## Tools Used

* Power BI
* DAX
* Bookmarks 
* Data Visualization

## Key Metrics

* Total Revenue
* Units Sold
* Average Order Value (AOV)
* Sales Contribution by Product
